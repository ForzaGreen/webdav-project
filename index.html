WebDAV Project

JS libraries:

This html imports:

  • jQuery
  • js-webdav-client

CORS:

Due to CORS problems, you need to open this page using the command:
$ google-chrome --disable-web-security
or:
$ chromium-browser --disable-web-security

Tools:

Chrome's Console: F12

Examples:

Ajax jQuery:

PROPFIND method:
$.ajax({ method: "PROPFIND", url:"http://localhost/webdav/ownCloudUser04/Documents/fichier6.txt"})

COPY method:
$.ajax({ method: "COPY", url:"http://localhost/webdav/sample.txt", headers:{Destination:"http://localhost/webdav/sample3.txt"} })

js-webdav-client:

Client declaration:
c = new nl.sara.webdav.Client("localhost/webdav", 0);

PROPFIND method:
c.propfind("sample.txt", function (status, data, headers) { console.log(status); console.log(data); console.log(headers); });

COPY method:
c.copy("sample.txt", function(status, data, headers){ }, "http://localhost/webdav/sample3.txt");
